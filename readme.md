# Mi Página HTML Semántica

Esta carpeta contiene una mini página personal construida con HTML semántico y CSS básico. Abre `index.html` directamente en tu navegador para verla. 
También puedes acceder a ella a través del servidor de GitHub en: https://davirigoyen.github.io/Mi-Pagina-Semantica/

## ¿Qué revisar?
- El HTML está estructurado con header, nav, main, article, aside, footer.
- Hay 1 imagen relevante de perfil en el <article> y otras dos secundarias (una en el <head> y otra en el <aside>).
- El CSS externo se carga correctamente desde `css/styles.css`.
- Los enlaces funcionan y las imagenes se muestran.
- Se utiliza el selector de atributo de CSS `a[target="_blank"]`.
- Se utilizan combinadores de CSS, entre ellos: `+ p`.
- Se utilizan las pseudo-clases de CSS: `a:link`, `a:visited` y `a:hover`.
- Se utilizan los pseudo-elementos de CSS: `::selection`, `::before`, `::after`, `::first-letter` y `::first-line`.
- Todo el contenido es claro y fácil de leer en todos los navegadores de PC de Escritorio y de teléfonos celulares.
